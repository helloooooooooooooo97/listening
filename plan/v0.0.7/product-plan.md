## 新追加：🔴 P0 — 音频详情统计面板

> **用户故事**：打开文章播放的歌词页面时，我想看到这篇文章相关的学习数据。

### 功能描述

在歌词全屏展开页面的**歌词上方或下方**，增加一个紧凑的统计信息条，显示当前音频的学习进度数据。

### 数据指标

```
┌─────────────────────────────────────────────────────┐
│  📊 文章统计                                        │
│                                                     │
│  🎯 进度 45%     ⏱ 已听 12分钟    📝 听写均分 78%   │
│  📖 总词数 523    ✅ 掌握 127词     📌 片段 5个       │
│  🔤 句子 48个     🏆 完成 1次                        │
│                                                     │
│  ┌───────────── 进度条 ─────────────────────────┐   │
│  │ ████████████████░░░░░░░░░░░░░░░ 45%           │   │
│  └──────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────┘
```

### 后端 API

**`GET /api/stats/audio-detail/{audio_id}`** — 返回单篇音频的详细学习数据

```json
{
  "audio_id": "fox-grapes",
  "title": "The Fox and the Grapes",
  "total_words": 115,
  "total_sentences": 8,
  "duration_seconds": 44.7,
  "known_words": 87,
  "listening_seconds": 720,
  "dictation_avg_score": 78.5,
  "dictation_count": 5,
  "completed": true,
  "clips_count": 3,
  "last_position": 30.2,
  "last_practiced": "2026-06-06"
}
```

### 交互设计

1. **折叠/展开**：统计条默认展开，可点击折叠为仅一行
2. **点击跳转**：点击「📌 片段 N 个」跳转到片段列表；点击「📝 听写均分」跳转到听写记录
3. **进度条**：显示当前播放位置 + 总体完成度
4. **实时更新**：播放时进度（已听分钟数）实时更新
5. **动画**：折叠/展开时平滑过渡
