# 产品计划 — 英语听力 App `v0.0.7`

> 继承自 [v0.0.6](../v0.0.6/product-plan.md) · 基于 [v0.0.6 评审报告](../v0.0.6/review-report.md)
>
> **v0.0.7 主题：音质体验 + 波形可视化 + 导入 UI**

## 目标

提升听力练习的核心体验，补齐课程导入功能，打磨 UI 细节。

---

## 范围

### 🔴 P0 — 必须交付

#### 1. 音频波形可视化

> **用户故事**：我想看到音频的波形，方便精确定位到想要听的位置。

- [ ] **波形组件**：Canvas 绘制音频波形（WAV 峰值为基础）
- [ ] **播放进度指示**：波形上叠加播放进度线
- [ ] **点击定位**：点击波形任意位置 seek 到对应时间
- [ ] **选词范围指示**：单词/句子在波形上的起止高亮
- [ ] **后端**：`GET /api/lessons/{id}/waveform` 返回归一化波形数据（如 1000 个采样点）

```
波形渲染方案：
1. 后端加载音频 → librosa 或 scipy 读取 → 下采样到 1000 点 → 返回 json
2. 前端 Canvas 绘制 → 进度线动画 → 点击 seek
3. 选词高亮：鼠标选中文本时对应波形区域变色
```

#### 2. 课程导入 UI

> **用户故事**：我想在 App 里直接上传音频文件，自动生成带词对齐的课程。（v0.0.5 未交付）

- [ ] **导入入口**：首页/音频页顶部「+ 导入」按钮
- [ ] **上传界面**：拖拽 MP3/WAV 或点击选择文件
- [ ] **元数据表单**：标题、副标题、难度等级
- [ ] **后台处理**：调用 WhisperX 转写 → 对齐 → 生成 JSON
- [ ] **进度反馈**：Toast 或进度条显示处理状态
- [ ] **完成后刷新**：课程列表自动更新

#### 3. 侧边栏收藏计数同步

> **问题**：收藏/取消收藏后侧边栏计数不立即更新（只在页面刷新后变）

- [ ] `favoritesStore` 的 `toggle` 返回新计数，Sidebar 响应式更新
