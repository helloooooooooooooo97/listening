# 产品计划 — 英语听力 App `v0.0.8`

> 继承自 [v0.0.7](../v0.0.7/product-plan.md)
>
> **v0.0.8 主题：课程导入 + 播放列表 + 学习提醒**

## 目标

补齐课程导入管线，增强播放列表管理，添加学习提醒功能。

---

## 范围

### 🔴 P0 — 必须交付

#### 1. 课程导入 UI
- [ ] 导入入口：首页/音频页顶部「+ 导入」按钮
- [ ] 上传界面：拖拽 MP3/WAV 或点击选择文件
- [ ] 元数据表单：标题、副标题、难度等级
- [ ] 后台处理：调用 WhisperX 转写 → 对齐 → 生成 JSON
- [ ] 进度反馈：Toast 或进度条显示处理状态
- [ ] 完成后刷新：课程列表自动更新

#### 2. 播放列表/队列
- [ ] 当前播放列表（正在播放/即将播放）
- [ ] 拖拽排序
- [ ] 添加到队列
- [ ] 播放历史回溯

### 🟡 P1 — 应该交付

#### 3. 学习提醒
- [ ] 每日学习目标设置（分钟数/课程数）
- [ ] 浏览器通知
- [ ] 连续学习天数统计增强

#### 4. 收藏批量管理
- [ ] 多选/全选
- [ ] 批量取消收藏
- [ ] 按日期/类型筛选

### 🟢 P2 — 有余力则做

#### 5. 统计图表增强
- [ ] 柱状图数值标签
- [ ] 鼠标悬停精确值
- [ ] 加载动画过渡

#### 6. 移动端响应式
- [ ] 侧边栏折叠为底部 Tab
- [ ] 播放栏适配小屏

## v0.0.7 → v0.0.8 对比

| 维度 | v0.0.7 | v0.0.8 |
|---|---|---|
| 课程导入 | 手动放 JSON | UI 上传 + WhisperX 自动转写 |
| 播放管理 | 单曲播放 | 播放列表 + 队列管理 |
| 学习提醒 | 无 | 每日目标 + 通知 |
