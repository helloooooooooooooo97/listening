# v0.1.3 产品计划 — 移动端体验 & 收尾打磨

**日期**: 2026-06-07
**主题**: 补齐移动端交互 + 播放体验收尾。

---

## 一、移动端手势

### 1.1 Transcript 滑动切句
- [ ] 监听 touch 事件：左右滑动切换上一句/下一句
- [ ] 滑动距离 > 80px 触发切换，带动画回弹
- [ ] 切换后自动 seek 到对应句子开始播放

### 1.2 移动端底部工具栏
- [ ] 移动端（< 768px）transcript 底部固定工具栏
- [ ] 按钮：⏪ 5s / ▶ 播放 / 10s ⏩ / 字号
- [ ] 毛玻璃半透明背景，不遮挡内容

### 1.3 双指缩放字号
- [ ] transcript 区域双指捏合调整字号
- [ ] 三档：小(14px) / 中(16px) / 大(18px)
- [ ] 变更时短暂显示当前档位提示

---

## 二、播放体验

### 2.1 播放速度记忆
- [ ] 每个音频独立记忆上次使用的播放速度
- [ ] 首次播放使用全局默认速度
- [ ] 速度切换时保存到 localStorage（按 lessonId）

### 2.2 睡眠定时器
- [ ] 设置页新增「睡眠定时」选项
- [ ] 选项：关闭 / 15分钟 / 30分钟 / 45分钟 / 60分钟
- [ ] 倒计时结束前 1 分钟音量渐弱
- [ ] 到时间自动暂停，显示 Toast 提示
- [ ] 播放栏显示剩余时间（定时器激活时）

### 2.3 连续播放完播提示
- [ ] 片段连续播放全部结束后 Toast 提示「N 个片段播放完毕」
- [ ] 自动恢复为普通播放模式
