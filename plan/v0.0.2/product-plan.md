# 产品计划 — 英语听力 App `v0.0.2`

## 目标

构建一个英语听力 App，将短音频课程转化为互动学习会话，支持句子播放、词级高亮、听写、跟读、词汇复习和进度追踪。

## 核心功能：片段收藏

> **核心理念**：用户可以在逐词高亮的文本中选中一段连续的词语（拖拽或 Shift+点击），将其保存为"片段"。收藏的片段可以单独循环播放，实现精准复习。不从零散单词出发，而是从语境片段出发，让复习始终携带上下文。

### 用户工作流

```
播放课程 → 听到感兴趣的段落 → 在文本中拖选词语范围 → 保存为片段
    ↓
复习模式 → 打开片段列表 → 点击播放 → 仅循环该片段（携带前后文）
```

### 为什么是「连续片段」而非「单个单词」

| 对比维度 | 单个单词收藏 | 连续片段收藏 |
| --- | --- | --- |
| 语境保留 | ❌ 丢失上下文 | ✅ 自然携带前后文 |
| 听力复习 | ❌ 零散播放 | ✅ 连续播放完整语义单元 |
| 口语模仿 | ❌ 无法跟读 | ✅ 可反复模仿完整短语/句子 |
| 记忆效率 | ❌ 孤立记忆 | ✅ 情景记忆（episodic memory） |

### 片段示例

```
选词范围：w-0012 ~ w-0023
文本内容："jumping as high as he could into the air"
时间范围：24.5s — 28.3s
所属课程：fox-grapes
标签：常用短语、动作描写
```

## 片段收藏详细设计

### 交互流程

```
1. 用户在 TranscriptView 中按下鼠标拖拽选中连续词语（或 Shift+点击首尾词）
2. 选中区域高亮显示（蓝色底色 + 虚线边框）
3. 弹出浮动操作栏：[保存为片段] [添加标签] [取消]
4. 保存后，片段加入 localStorage 片段列表
5. 复习模式：从片段列表进入，仅播放片段时间范围，自动循环
```

### 片段数据结构

```typescript
interface AudioClip {
  id: string;           // 唯一标识，如 "clip-001"
  lessonId: string;     // 所属课程 ID
  startWordId: string;  // 起始词 ID，"w-0012"
  endWordId: string;    // 结束词 ID，"w-0023"
  startTime: number;    // 起始时间（秒）
  endTime: number;      // 结束时间（秒）
  text: string;         // 片段文本
  note: string;         // 用户注释/标签
  createdAt: string;    // 创建时间 ISO
}
```

### 复习模式

- 片段列表页：展示所有收藏的片段，按课程/时间分组
- 逐片段播放：点击片段 → 跳转到对应音频时间 → 循环播放该片段
- 连续复习：按顺序自动播放所有片段（每个片段循环 N 次后进入下一个）
- 快速定位：点击片段可跳转到原课程中的位置
