# v0.1.2 产品计划 — 播放详情页交互完整体验

**日期**: 2026-06-07
**主题**: 全面补全播放详情页的交互能力，覆盖文本操作、片段管理、听写复练、播放增强、移动端手势。

---

## 一、当前状态

播放详情页现有：
- 逐句 transcript + 词级高亮 + 片段颜色标注
- 右侧可折叠 tabs（片段 / 听写 / 收藏）
- hover 联动高亮、点击跳播

**缺失的能力**：
- 文本上没有任何右键/双击/键盘操作
- 片段只有查看和播放，不能编辑、删除、排序
- 听写记录只能看，不能从这里直接复练
- 播放控制缺少 A-B 循环、音量、快进快退
- 移动端没有手势支持
- 没有深色/浅色模式之外的显示偏好

---

## 二、详细任务

### 🔴 P0 — 文本交互基础能力

#### 2.1 词级右键菜单
- [ ] 右键单词弹出上下文菜单（不干扰拖拽选区）
- [ ] 菜单项：🔊 播放此词 / ✓ 标记掌握 / ♥ 收藏 / 📖 查词典(新标签页打开)
- [ ] 菜单位置跟随光标，不超出视口边界
- [ ] 点击菜单外或 Esc 关闭

#### 2.2 双击单词操作
- [ ] 双击 transcript 中的单词 → 标记为「已掌握」（toggle）
- [ ] 已掌握单词颜色变为绿色，未掌握保持默认
- [ ] 双击时给出微小的确认动画（单词短暂缩放 + 颜色过渡）

#### 2.3 键盘导航
- [ ] `↑` / `↓` 键在 transcript 句子间移动焦点
- [ ] `Enter` 播放当前聚焦的句子
- [ ] `Space` 播放/暂停（全局已有，需确保在 transcript 区域不冲突）
- [ ] 当前聚焦句子显示微妙的左侧指示条
- [ ] `Esc` 取消聚焦，回到自动跟随模式

#### 2.4 自动滚动开关
- [ ] transcript 区域右上角添加 📌 按钮（切换自动滚动）
- [ ] 关闭自动滚动后，播放时 transcript 不再自动跟随
- [ ] 打开后立即滚动到当前播放位置
- [ ] 按钮状态用 tooltip 提示

---

### 🟡 P1 — 片段管理增强

#### 3.1 侧边栏片段操作
- [ ] 每个片段项右侧显示更多按钮（···），点击弹出操作菜单
- [ ] 操作项：✏️ 编辑备注 / 🎨 改颜色 / 🗑 删除
- [ ] 编辑备注：inline 输入框，Enter 保存，Esc 取消
- [ ] 改颜色：弹出 6 色小圆点，点击即改
- [ ] 删除：确认弹窗「确定删除片段？」+ 收缩动画

#### 3.2 片段排序 & 筛选
- [ ] tab 栏下方添加排序按钮：按时间 / 按日期
- [ ] 按时间排序：片段在音频中的先后顺序（默认）
- [ ] 按日期排序：最近创建的在前
- [ ] 排序按钮显示当前排序状态

#### 3.3 导出片段
- [ ] 片段 tab 底部添加「导出」按钮
- [ ] 点击导出为纯文本：每行一个片段 `[00:23] text`
- [ ] 复制到剪贴板 + Toast 提示「已复制 N 个片段」

#### 3.4 播放所有片段
- [ ] 片段 tab 顶部添加「▶ 连续播放」按钮
- [ ] 点击后按时间顺序依次播放所有片段
- [ ] 播放中按钮变为「⏸ 停止」
- [ ] 全部播放完毕后恢复

### 🟡 P1 — 听写复练

#### 4.1 侧边栏快捷复练
- [ ] 每条听写记录右侧添加「🔁 复练」按钮
- [ ] 点击 → 关闭详情页 → 进入听写模式并定位到该句子
- [ ] 听写记录按句子分组显示，可折叠/展开

#### 4.2 错句筛选
- [ ] tab 栏下方添加筛选按钮：全部 / 错句(<80%) / 未练
- [ ] 错句模式下仅显示低于 80% 的句子记录
- [ ] 底部「🔁 复练全部错句」按钮（仅错句模式显示）

#### 4.3 进度可视化
- [ ] 每句显示迷你进度条（已练次数颜色深浅）
- [ ] 从未练过的句子显示为虚线边框 + 「未练」标记

### 🟢 P2 — 播放增强

#### 5.1 A-B 循环
- [ ] transcript 区域顶部添加 `[A]` `[B]` 按钮
- [ ] 点击 [A] 标记循环起点（当前播放位置），[A] 高亮
- [ ] 点击 [B] 标记循环终点，[B] 高亮，开始循环
- [ ] 循环中的区域在波形上高亮显示
- [ ] 再次点击 [A] 或 [B] 取消循环
- [ ] 快捷键：`[` 设 A 点，`]` 设 B 点

#### 5.2 快进快退
- [ ] transcript 底部添加浮动的快退 5s / 快进 10s 按钮
- [ ] 快捷键：`←` 退 5s，`→` 进 10s（仅当键盘导航未激活时）
- [ ] 点击时显示 ± 时间指示器（短暂弹出后消失）

#### 5.3 音量控制
- [ ] 播放栏右侧添加音量图标 🔈
- [ ] 点击弹出音量滑块（垂直 slider）
- [ ] 快捷键：`Ctrl+↑` / `Ctrl+↓` 调整音量
- [ ] 音量偏好保存到 localStorage

#### 5.4 睡眠定时器
- [ ] 设置中可开启睡眠定时：15/30/45/60 分钟
- [ ] 倒计时结束前 1 分钟音量渐弱
- [ ] 到时间自动暂停播放
- [ ] 播放栏显示剩余时间（如果开启了定时器）

---

### 🟢 P2 — 显示偏好 & 移动端

#### 6.1 Transcript 显示偏好
- [ ] transcript 顶部添加显示选项按钮（齿轮图标）
- [ ] 弹出下拉：字号（小/中/大）、行距（紧凑/舒适）、显示行号/时间戳（开关）
- [ ] 偏好保存到 localStorage
- [ ] 字号和行距立即生效

#### 6.2 移动端手势
- [ ] transcript 区域左右滑动 → 切换上一句/下一句（带惯性动画）
- [ ] 双指缩放 → 调整 transcript 字号
- [ ] 长按单词 → 弹出移动端适配的上下文菜单

#### 6.3 移动端底部工具栏
- [ ] 移动端 transcript 底部固定工具栏
- [ ] 按钮：A-B 循环、快退 5s、快进 10s、显示选项
- [ ] 工具栏半透明毛玻璃背景
