## 已实现功能

### P0 文本交互
- 右键单词弹出上下文菜单：播放此词 / 标记掌握 / 收藏 / 查词典(Cambridge)
- 双击单词快速切换掌握状态 + 颜色过渡
- 键盘 ↑↓ 句子间导航，Enter 播放，Esc 取消
- 自动滚动开关（📌 按钮）

### P1 片段管理
- 侧边栏内编辑片段备注（inline 输入，Enter 保存）
- 侧边栏内改颜色（6色小圆点行）
- 删除片段（带 Toast 反馈）
- 排序切换：按时间 / 按日期
- 「导出」按钮 → 复制所有片段到剪贴板
- 「连续播放」按钮 → 按时间顺序播放全部片段

### P1 听写复练
- 低分(<80%)听写记录显示「复练」按钮

### P2 播放增强
- A-B 循环：设 A/B 点，显示时间范围
- 快退 5s / 快进 10s 按钮
- 音量滑块（hover 展开）
- 字号切换：A⁻ / A / A⁺
