## 二、任务

### 🔴 P0 — 播放控制增强

#### 2.1 音量控制
- [ ] 播放栏音量 hover 展开 slider，图标根据音量切换（🔇 🔉 🔊）
- [ ] 音量变化持久化到 localStorage，启动时恢复
- [ ] 快捷键：`Ctrl+↑` / `Ctrl+↓` 调音量

#### 2.2 播放速度按音频记忆
- [ ] 每个音频独立保存播放速度（`speed:${lessonId}`）
- [ ] 切入音频时自动恢复记忆速度
- [ ] 首次使用全局默认值

---

### 🟡 P1 — 侧边栏收藏操作

#### 3.1 收藏 tab 改进
- [ ] 收藏单词点击 → 跳转到音频中该词位置
- [ ] 收藏片段/单词可加入队列（➕ hover 按钮）
- [ ] 按类型分组显示（音频/片段/单词）
- [ ] 每组带分组标题和计数

---

### 🟡 P1 — Transcript 快捷操作

#### 4.1 双击句子行
- [ ] 双击句子行 → 开启听写模式并定位到该句
- [ ] 双击时句子行短暂高亮反馈

#### 4.2 全局快捷键提示
- [ ] 按 `?` 弹出快捷键帮助面板
- [ ] 列出：Space(播放/暂停) ↑↓(导航句子) Ctrl+↑↓(音量) Esc(退出)

---

### 🟢 P2 — 睡眠定时器

#### 5.1 睡眠定时

- [ ] 选项：15/30/45/60 分钟
- [ ] 到时间自动暂停音频
