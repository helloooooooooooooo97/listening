# 产品计划 — 英语听力 App `v0.0.3`

> 继承自 [v0.0.2](../v0.0.2/product-plan.md) · 基于 [v0.0.2 评审报告](../v0.0.2/review-report.md)
>
> **v0.0.3 主题：精听控制 + 体验打磨**

## 目标

在 v0.0.2 的核心闭环（选课→播放→收藏→复习）基础上，补齐精听控制能力，打磨交互细节，让产品从"能用"变为"好用"。

## v0.0.3 范围

### 🔴 P0 — 必须交付

#### 1. 逐句导航

> **用户故事**：我在听课程时，遇到听不懂的句子，想反复听这一句。

- [ ] **上一句/下一句按钮**：底部播放栏增加 ⏮ ⏭ 按钮
- [ ] **跳转行为**：点击后跳转到当前句/下一句的 `start` 时间
- [ ] **自动播放**：跳转后自动开始播放（如果当前正在播放中）
- [ ] **视觉反馈**：跳转时右侧文本自动滚动到对应句子

#### 2. 循环模式

> **用户故事**：我想让当前句子自动重复播放，直到我听懂为止。

- [ ] **循环模式切换**：底部播放栏增加循环模式按钮
  - `📜 全部` — 顺序播放全部句子
  - `🔂 单句` — 重复当前句子
  - `🔁 片段` — 片段模式（已实现）
- [ ] **单句循环逻辑**：播放到当前句 `end` 时自动 seek 回 `start`
- [ ] **当前句判定**：基于 `currentTime` 实时判定所在句子索引

#### 3. 键盘快捷键

> **用户故事**：我不想用鼠标点来点去，希望像音乐播放器一样用键盘操作。

- [ ] **全局快捷键**：
  | 按键 | 行为 |
  |---|---|
  | `Space` | 播放/暂停 |
  | `←` | 后退 5 秒 |
  | `→` | 前进 5 秒 |
  | `↑` | 上一句 |
  | `↓` | 下一句 |
  | `R` | 切换循环模式 |
  | `1-5` | 切换播放速度（0.5x-1.5x） |
  | `S` | 保存当前句子为片段 |
- [ ] **焦点管理**：快捷键在输入框聚焦时不触发

#### 4. 清理技术债

- [ ] 删除 `hooks/useAudioPlayer.ts`（已被 `audioStore` 替代）
- [ ] 删除 `hooks/useClips.ts`（已被 `clipsStore` 替代）
- [ ] 删除 `frontend/src/assets/` 中的未使用文件

### 🟡 P1 — 应该交付

#### 5. Toast 通知系统

> **用户故事**：我保存片段后，想知道是否成功。

- [ ] **Toast 组件**：右上角弹出，3秒自动消失
- [ ] **触发场景**：保存片段 ✅、删除片段 ✅、更新标签 ✅、切换模式
- [ ] **动画**：从右侧滑入 + 淡出

#### 6. 拖拽选词优化

> **用户故事**：拖拽选词时，文本不应该被浏览器选中变成蓝色。

- [ ] **禁止浏览器默认选中**：`user-select: none` + `preventDefault`
- [ ] **选中态视觉**：拖拽经过的词用浅蓝底色 + 虚线边框
- [ ] **取消选中**：点击空白区域或按 `Escape`

#### 7. 片段管理增强

> **用户故事**：我收藏了20个片段，想快速找到某个或按课程查看。

- [ ] **排序方式**：按时间（最新/最早）、按时长、按课程分组
- [ ] **批量删除**：编辑模式下多选删除
- [ ] **搜索高亮**：搜索关键词在片段文本中高亮显示

#### 8. 播放位置记忆

> **用户故事**：我刷新页面后，希望能继续上次听的位置。

- [ ] **记录位置**：播放过程中每5秒记录当前课程ID + 时间到 localStorage
- [ ] **恢复提示**：下次打开同一课程时，在文本区顶部显示"跳转到上次位置"
- [ ] **自动恢复**：可选，点击后跳转

#### 9. 过渡动画

> **用户故事**：切换课程或片段时，界面变化太突兀。

- [ ] **卡片选中态**：左侧卡片选中时背景色渐变（150ms ease）
- [ ] **面板内容**：切换课程时内容淡入淡出（200ms）
- [ ] **播放按钮**：▶ ↔ ⏸ 切换时缩放动画

- [ ] 播放栏按钮和文字适配小屏幕

## v0.0.2 → v0.0.3 对比

| 维度 | v0.0.2 | v0.0.3 |
|---|---|---|
| 句子导航 | 只能点击跳转 | 上下句按钮 + 键盘快捷键 |
| 循环控制 | 仅片段循环 | 全部/单句/片段三种模式 |
| 操作反馈 | 无 | Toast通知 |
| 状态记忆 | 刷新丢失 | 播放位置记忆 |
| 片段管理 | 列表浏览 | 排序/筛选/搜索高亮 |
| 动效 | 无 | 选中/切换/按钮过渡动画 |
| 代码质量 | 残留废弃文件 | 清理hooks/ |
